Short Biography

I am a software engineer at IQGeo. I obtained my PhD in Computer Science at the same university by investigating the computational aspects of plants at Ghent University. Moreover, I graduated as Master of Science in Electrical Engineering with main subject Communication and Information Technology (CIT) at the same in July 2017. I am passionate about science, technology, photography, languages and travel. Sometimes, I work on personal projects. You can find them here. A five word description of me would be: enthusiastic, observant, perfectionistic, driven and minimalistic.

About this Website

This is a mainly a static site, created with Hugo. Static means it only uses HTML5, CSS and (mimited) JavaScript to display the content on this site, no server side language such as PHP is used. The photo galleries and posts that include mathematics require a browser with JavaScript support. The photo galleries are created using free (for non-commercial use) and open source frameworks, lightGallery and Isotope. KaTeX provides support for high quality mathematical expressions. Please mind that you will need a recent browser to view this site in its full glory.
