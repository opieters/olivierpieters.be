To provide additional IO connectivity, we will use the ESP8266-based Adafruit Huzzah. This board is not programmable using a USB connection as an Arduino, so we will need a Serial adapter. We will use a [CP2102 based adapter](CP2102-Amazon). Tu use this board, first download the CP2102 driver from [Silicon Labs](https://www.silabs.com/products/mcu/Pages/USBtoUARTBridgeVCPDrivers.aspx#mac). Afterwards connect the pins as follows:


| pin CP2102 | pin Huzzah |
|:----------:|:----------:|
| DTR        |            |
| RX         | TX         |
| TX         | RX         |
| VCC        | VBAT       |
| CTS        | GND        |
| GND        | GND        |

Now we have correctly entered the required hardware setup. The IDE of choice will be the PlatformIO toolchain. An installation guide can be found on the [website][PlatformIO]. If you already installed Atom, you can also [install the plugin][PlatformIO-plugin].
