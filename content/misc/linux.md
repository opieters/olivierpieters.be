Some useful how-to's and tricks regarding Linux.

## Setting up Ubuntu the right way

1. Download the Unity version of Ubuntu from the official website.
2. Update: `sudo apt-get update; sudo apt-get upgrade;`
3. Install Gnome: `sudo apt-get install gnome-desktop-environment` (you may add `-core` or `-extra` to install only specific components)
4. Change the default UI from the login screen (small icon near password box)
